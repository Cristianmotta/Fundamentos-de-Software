Practica de Fundamentos de css en Platzi

ClASE:

Una clase en HTML y CSS es un nombre que se asigna a un elemento para aplicarle estilos o comportamientos específicos. Se define con el atributo class y se llama en CSS con un punto (.).

GRID:

CSS Grid Layout es un sistema de diseño basado en filas y columnas, ideal para crear estructuras más complejas como páginas completas o galerías.Permite colocar elementos con precisión en una cuadrícula

FLEXBOX:

Es un sistema de diseño en CSS que permite organizar y alinear elementos fácilmente dentro de un contenedor.
